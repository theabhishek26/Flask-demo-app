# Flask-demo-app

A simple **Flask based web application** that display styled page 
This project demonstrate :
- Basic of Flask routing 
- HTML templating
- CSS styling

---

# Features
- Flask backend serving HTML templates
- Static CSS  for the desogn 
- Rsponsive
